Ramel HR — Style Guide

Bootstrap 5 + style.css usage reference

1. Setup

Add these two lines in your HTML <head>, in this order:

<!-- 1. Bootstrap 5 CSS -->
<link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- 2. Ramel custom styles (AFTER Bootstrap) -->
<link href="style.css" rel="stylesheet">

<!-- Bootstrap JS (before closing body) -->
<script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"></script>

Rule: Always load style.css after Bootstrap so Ramel styles override Bootstrap defaults cleanly.

2. Layout Structure

Wrap your entire app in .app-wrapper. The sidebar uses fixed positioning; the main content uses margin-left: var(--sidebar-width) automatically.
